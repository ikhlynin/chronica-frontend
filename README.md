## Quick Start

1. **Clone the repository:**
`git clone <your-repo-url>`
`cd chronica-frontend`

2. **Install dependencies:**
`yarn install`

3. **Run in development mode:**
`yarn dev`

4. **Build for production:**
`yarn build`

5. **Preview the production build locally:**
`yarn preview`
